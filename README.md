# Melody

## Tech/tools used - 

- Typescript
- NextJS
- Tailwind CSS
- Recoil

## Getting Started

First, run the development server:

1. Fork the repo.
2. Clone the forked repo.
3. 
```sh
cd melody
cp .env.example .env
npm install
```
4. Fill the .env files with your personal env variables.
5.  
```sh
npm run dev
```
The development server is now running on [http://localhost:3000](http://localhost:3000).
